  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. [de returnerer boolean verdier. sjekk om de er true eller false ved å bruke if etter const]

  6. How does the Math.random() function work, and what can it be used for?

    7. [math.random() returnerer et tilfeldig tall mellom 0 (inkludert) og 1 (ekskludert). man kan bruke math.floor() og math.ceil() for å nå høyere eller lavere verdier. det kan brukes for å lage tilfeldige tall til f.eks spill]
